singlestoredb.connection.Cursor¶

class singlestoredb.connection.Cursor(connection: singlestoredb.connection.Connection, cursor: Any, driver: singlestoredb.drivers.base.Driver)¶

Database cursor for submitting commands and queries.

This object should not be instantiated directly. The Connection.cursor method should be used.

__init__(connection: singlestoredb.connection.Connection, cursor: Any, driver: singlestoredb.drivers.base.Driver)¶

Call Connection.cursor instead.
