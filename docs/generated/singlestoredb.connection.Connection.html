singlestoredb.connection.Connection¶

class singlestoredb.connection.Connection(**kwargs: Any)¶

SingleStoreDB connection.

Instances of this object are typically created through the singlestoredb.connect() function rather than creating them directly. See the connect() function for parameter definitions.
