FileSpace.upload_file(local_path: str | PathLike[str] | IOBase, path: str | PathLike[str], *, overwrite: bool = False) → FilesObject¶

Upload a local file.

Parameters:

  • local_path (Path or str or file-like) – Path to the local file or an open file object

  • path (Path or str) – Path to the file

  • overwrite (bool, optional) – Should the path be overwritten if it exists already?
